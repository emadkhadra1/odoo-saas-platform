  • Settlements
  • Performance Dashboard

Construction

  • Progress Bills
  • Project Reports
  • Budgeting

مبني على Jadeer Real Estate، لإدارة دورة العقار من CRM إلى الحجز والدفعات والمشاريع.

  • Lessons
  • Exams
  • Assignments
  • Live Sessions

Demo Roadmap

خطة تجهيز الديموهات

01تجهيز Master DB وفيه `odoo_saas_manager` + website + CRM leads.
02إنشاء template DB لكل قطاع أو template عام + modules حسب الخطة.
03تنظيف dependencies واختبار install لكل vertical على إصدار Odoo المستهدف.
04إضافة demo data: عميل، مشروع، فاتورة، دفعة، مستخدمين، وسيناريو تشغيل كامل.
05ربط صفحات الموقع بأزرار: طلب تجربة، إنشاء tenant، ومتابعة الاشتراك.
