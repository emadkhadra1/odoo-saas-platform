# Odoo SaaS Marketing Website

Install this addon in the Odoo database that hosts the public marketing website.

The addon adds the public route:

```text
/saas-platform
```

It serves the interactive Arabic SaaS presentation from `static/preview/`, including:

- Sector solution cards and features.
- Saudi Riyal pricing.
- Interactive SaaS Manager backend preview.
- Tenant creation simulation.
- Plans, subscriptions, payments, logs, and onboarding previews.

This module currently preserves the standalone preview exactly. It can later be converted into native Odoo Website Builder snippets.
